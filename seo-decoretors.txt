# دیکوریتور پرامت SEO حرفه‌ای: +++seoplusai

## نحوه فراخوانی:
```
+++seoplusai(domain=URL, format=code|text, images=boolean, tone=formal|friendly|technical, lang=FA|EN|...)
```

### پارامترهای ورودی:
- **domain:** آدرس وب‌سایت هدف (مثال: volesta.ir)
- **format:** نوع خروجی (code برای HTML کامل، text برای متن ساده)
- **images:** true/false برای شامل کردن پرامت‌های تصویر
- **tone:** تون نگارش (formal=رسمی، friendly=دوستانه، technical=تخصصی)
- **lang:** زبان محتوا (FA=فارسی، EN=انگلیسی)
